# Without Exif

Бот для удаления Exif данных из фото.


![logo](source/screenshots/logo.jpg)

# Запуск

Установи нужные библиотеки:
```bash
pip3 install aiogram
```

```bash
pip3 install Pillow
```

Установи `API_KEY` в `config.py`:
```python
# Config.py

API_KEY = '62843...fO70g'
```

Запусти бота (`run.py`):
```bash
cd WithoutExif
```
```bash
python3 run.py
```
